# BingeBoard

A minimal OTT launcher to replace cluttered browser shortcuts with a clean, customizable hub.

## Features
- Add your own OTT shortcuts (name + URL)
- Pin important apps to the top
- Delete shortcuts easily
- Clean grid layout
- Data saved in browser (localStorage)
- No login, no backend

## Usage
1. Open `index.html` in your browser  
2. Add your streaming apps  
3. Click any tile to open it  

## Tech
- HTML
- CSS
- JavaScript (vanilla)
- localStorage

## Goal 
### Keep all your streaming apps in one simple place without cluttering your browser.
